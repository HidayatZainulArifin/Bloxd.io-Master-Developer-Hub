Quick Time Events

Master Quick Time Events (QTEs)

1. Usage Patterns

Triggering a QTE from server-side and handling results in the callback:

// 1. Starting a QTE
const qteId = api.addQTE(playerId, {
    type: "progressBar",
    parameters: {
        progressDecreasePerTick: 0.075,
        progressPerClick: 7,
        canFail: false,
        description: [{ str: "Click repeatedly to complete!" }],
        clickIcon: "fa-solid fa-computer-mouse",
    },
})

// 2. Handling the Result (In World Code)
onPlayerFinishQTE(playerId, qteId, result) {
    if (result) {
        // Success Logic
    } else {
        // Failure Logic
    }
}

// 3. Utility Methods
api.deleteQTE(playerId, qteId) // Cancel active QTE
const hasQTE = api.hasActiveQTE(playerId) // Check if player is in a QTE

2. QTE Types & Parameters

progressBar

Click rapidly to fill a progress bar before it drains to zero.

ParameterTypeDefault
progressStartValuenumber30
progressDecreasePerTicknumber0.075
progressPerClicknumber5
canFailbooleanfalse
clickIconstring"fa-solid fa-computer-mouse"

timedClick

A prompt appears and the player must click before the timer runs out.

ParameterTypeDefault
timeWindownumber (ms)3000
showTimerbooleantrue
breatheCenterbooleanfalse
scale / rotationnumber1 / 15

gravityBar

Hold click to push catch zone up, release to let it fall. Overlap the mover to gain progress.

ParameterTypeDefault
catchZoneSizenumber (0-1)0.25
moverSpeednumber3
gravity / riseSpeednumber1 / 1.5
progressGainPerSecondnumber8

precisionBar

Click when the marker is within the green success zone in the center.

ParameterTypeDefault
speednumber0.5
successZoneSizenumber (0-1)0.15
labelCustomText"Click when inside green zone"

rhythmClick

Click when a shrinking outer circle aligns with the fixed inner circle.

ParameterTypeDefault
requiredSuccessesnumber5
shrinkDurationMsnumber1200
toleranceFractionnumber0.15
maxMissesnumber3
