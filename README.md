# 🚀 Bloxd.io Master Developer Hub

Welcome to the **Bloxd.io Master Developer Hub**! This is the most comprehensive technical documentation site for developers looking to create custom games, mechanics, and game modes within the Bloxd.io universe using World Code and Code Blocks.

---

## 🛠️ Project Overview
This project serves as a "Master Library" for Bloxd.io coders. It translates complex technical data from the official API into a clean, searchable, and beginner-friendly web interface. Whether you are using Notepad++ or a mobile device, this hub is built to help you find IDs and functions instantly.

### ✨ Key Features:
- **Fast Search Engine:** Find any ID or function instantly across 1000+ entries.
- **Developer UI:** Clean dark-mode interface inspired by VS Code.
- **One-File System:** Lightweight documentation files that work offline.
- **Full API Access:** Includes everything from basic blocks to advanced QTE logic.
