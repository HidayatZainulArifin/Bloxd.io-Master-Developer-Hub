1. API Methods

Use these functions to modify the options during gameplay:

// Set a single option
api.setClientOption(playerId, "speedMultiplier", 2)

// Get current value of an option
const val = api.getClientOption(playerId, "maxHealth")

// Set multiple options at once
api.setClientOptions(playerId, { "canChange": false, "invincible": true })

// Reset to default
api.setClientOptionToDefault(playerId, "jumpAmount")
