🚀 Code API Master Reference

📌 Essential Basics

  • Player Identity: Use myId or playerId for the player running the code.
  • Coordinates: thisPos holds the [x, y, z] position of the code block.
  • Debugging: Use api.log("text") to send info to the F8 console.
  • Comments: Use /* catat */. Note: // is NOT supported.
📍 Position & Players
MethodDescription
api.getPosition(id)Get [x, y, z] of player.
api.setPosition(id, x, y, z)Teleport player to location.
api.getPlayerIds()List all online players.
❤️ Health & Combat
MethodDescription
api.getHealth(id)Check current health (0-100).
api.setHealth(id, value)Instantly set health level.
api.applyHealthChange(id, amt)Damage (-amt) or Heal (+amt).
🌍 Blocks & World
MethodDescription
api.setBlock(x, y, z, "Name")Place a single block.
api.setBlockRect(pos1, pos2, "Name")Fill an area with blocks.
api.getBlock(x, y, z)Check what block is at coordinates.
