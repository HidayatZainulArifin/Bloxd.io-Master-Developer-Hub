🚀 Code API Master Reference

📌 Essential Basics

  • Player Identity: Use myId or playerId to refer to the person running the code.
  • Coordinates: thisPos holds the [x, y, z] location of the current code block.
  • Debugging: Use api.log("message") to see info in the F8 console.
  • Comments: ONLY use /* my comment */. The // symbol is NOT supported and will break your code.
  • Time: Use Date.now() to get the current time in milliseconds.

🛠️ API Function Categories

Click a category below to see the available functions:

📍 Player & Position
MethodDescription
api.getPosition(id)Get the [x, y, z] position of a player.
api.setPosition(id, x, y, z)Teleport a player to specific coordinates.
api.getPlayerIds()Get a list of every player currently in the game.
api.isMobile(id)Check if the player is using a Phone or PC.
api.setCameraDirection(id, [x,y,z])Force the player to look in a specific direction.
❤️ Health & Combat
MethodDescription
api.getHealth(id)Check player health (usually 0 to 100).
api.setHealth(id, value)Instantly change player health.
api.applyHealthChange(id, amt)Damage (-amt) or Heal (+amt) a player.
api.killLifeform(id)Instantly kill a player or a mob.
api.getShieldAmount(id)Check how much armor shield a player has.
api.forceRespawn(id)Force the player to go back to spawn.
🌍 Blocks & World
MethodDescription
api.setBlock(x, y, z, "Name")Place a single block at a position.
api.setBlockRect(pos1, pos2, "Name")Fill a whole area with blocks.
api.getBlock(x, y, z)Get the name of the block at that position.
api.getBlockSolidity(x, y, z)Check if a block is solid (can walk on it).
api.raycastForBlock(pos, dir)Find the block a player is looking at.
🎒 Items & Inventory
MethodDescription
api.giveItem(id, "Name", count)Add items to player's inventory.
api.hasItem(id, "Name")Check if player has a specific item.
api.clearInventory(id)Remove everything from player's inventory.
api.createItemDrop(x, y, z, "Name")Spawn a floating item on the ground.
api.getHeldItem(id)Check what item the player is currently holding.
🖥️ UI & Messaging
MethodDescription
api.sendMessage(id, "text")Send a private chat message to a player.
api.broadcastMessage("text")Send a message to EVERYONE in the world.
api.sendHitmarker(id, isCrit)Show the "X" hit effect on a player's screen.
api.progressBarUpdate(id, 0.5, 1000)Show a progress bar on the bottom right.
api.setDirectionArrow(id, "id", pos)Show an arrow pointing to a location.
👾 Mobs & Physics
MethodDescription
api.attemptSpawnMob("Type", x, y, z)Spawn a pig, cow, zombie, etc.
api.applyImpulse(id, x, y, z)Push a player/entity in a direction.
api.setVelocity(id, x, y, z)Instantly set the movement speed/direction.
api.getMobIds()Get a list of all active mobs in the world.
api.despawnMob(mobId)Remove a mob from the world.

💡 Ready-to-Use Examples

Copy and paste these snippets into your Code Blocks:

/* 1. Healing Pad: Heal player when they stand on this block */
onBlockStand = (playerId, x, y, z, blockName) => {
    if (x == thisPos[0] && z == thisPos[2]) {
        api.applyHealthChange(playerId, 5);
        api.sendMessage(playerId, "You are being healed!");
    }
}

/* 2. Super Jump: Launch player into the air */
api.setVelocity(myId, 0, 15, 0);

/* 3. Global Announcement */
api.broadcastMessage("A new round is starting!", { color: "orange" });
