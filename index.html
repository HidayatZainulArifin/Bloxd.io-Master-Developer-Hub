API Reference (Basic)

FunctionUsage
api.getPosition(myId)Gets the current player's position.
api.sendMessage(myId, "hello")Sends a private chat message.
api.broadcastMessage("info")Sends a message to everyone.
api.setBlock(x, y, z, "Dirt")Places a block instantly.

Snippet Example:

// Disable player gravity (high jump)
api.setVelocity(myId, 0, 15, 0)

Callbacks (Events)

Code that runs automatically during specific events:

  • onPlayerJoin: Runs when a person joins the server.
  • onPlayerChat: Runs when someone types in chat.
  • onBlockStand: Runs when a player steps on a specific block.
  • onPlayerKilledOtherPlayer: Runs when a kill occurs.
